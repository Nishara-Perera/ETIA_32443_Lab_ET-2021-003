## Student Information

## Objective
To set up the embedded development environment, practice AVR register-level programming using Arduino Uno, and use GitHub for version control.

## Activities Completed

- Activity 01: LED blinking using AVR registers (DDRB, PORTB)  
- Activity 02: Sequential LED control using PORTC and bitwise operations  
- Activity 03: Push button input control using PIND  

## Conclusion
This lab demonstrated the successful setup of embedded development tools, AVR register-level programming, and basic GitHub version control.
